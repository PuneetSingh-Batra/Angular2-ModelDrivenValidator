CLI Setup:
    1. npm install -g angular-cli [cli install]
    2. ng new projectName [new project greate]
    3. cd projectName [enter in project]
    4. ng serve
    5. npm install typescript -g [typescript install - optional]
    6. ng generate component testComponent [create a new component]

Update node version:
   http://tecadmin.net/upgrade-nodejs-via-npm/

   For latest node update: sudo n stable
